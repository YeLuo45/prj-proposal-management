# PRD: android-hello GitHub Actions CI

## 1. 概述
为 android-hello 配置 GitHub Actions CI，实现代码推送后自动构建 APK 并上传到 Releases。

## 2. 功能需求

### 2.1 CI 触发条件
- push 到 main 分支
- pull request 到 main 分支
- 创建 git tag 时

### 2.2 构建任务
- 检出代码
- 设置 JDK 17
- 配置 Gradle
- 执行 assembleDebug 构建
- 上传 APK 到 GitHub Releases（仅 tag push 时）

### 2.3 构建产物
- devDebug APK
- stagingDebug APK
- prodRelease APK

## 3. 技术方案
- GitHub Actions workflow
- gradle.properties 配置 CI 环境
- actions/upload-artifact 上传构建产物
- softprops/action-gh-release 上传 Release

## 4. 验收标准
- [ ] push 到 main 自动触发构建
- [ ] 构建成功生成 APK
- [ ] 创建 tag 自动发布 Release
- [ ] PR 构建验证通过

## 5. 依赖
- 无新增项目依赖
