# PRD: android-hello 用户交互功能

## 1. 概述
为 android-hello 添加用户交互组件（Button、EditText），实现简单的输入-反馈功能。

## 2. 功能需求

### 2.1 布局改进
- 使用 ConstraintLayout 组织界面
- 添加 TextView 显示欢迎文字
- 添加 EditText 供用户输入名字
- 添加 Button 点击后显示个性化问候

### 2.2 交互逻辑
- 用户在 EditText 输入名字
- 点击 Button 后，TextView 显示 "你好, [名字]!"
- 空输入时提示"请输入名字"

## 3. 技术方案
- ConstraintLayout 替代原生 TextView
- View Binding 绑定 UI 组件
- Button onClickListener 处理点击
- InputType 设置 EditText 为纯文本

## 4. 验收标准
- [ ] 界面显示欢迎文字、输入框、按钮
- [ ] 输入名字后点击按钮显示个性化问候
- [ ] 空输入时有提示
- [ ] APK 成功构建

## 5. 依赖
- androidx.constraintlayout:constraintlayout:2.1.4（已添加）
