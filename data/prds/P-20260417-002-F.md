# PRD: android-hello Material Design UI

## 1. 概述
为 android-hello 引入 Material Design 3 组件，提升界面美观度和用户体验。

## 2. 功能需求

### 2.1 Material Components 替代原生组件
- MaterialToolbar 替代普通标题栏
- MaterialButton 替代普通 Button
- TextInputLayout + TextInputEditText 替代 EditText
- MaterialCardView 包装问候语显示区域

### 2.2 主题升级
- 应用 Material3 主题（DayNight 暗色模式支持）
- 自定义主题颜色：Primary #3F51B5, Secondary #FF4081
- 统一字体和间距规范

### 2.3 界面微调
- 添加 Toolbar 显示应用标题
- 输入框使用 outlinedTextInputLayout 样式
- 按钮使用 MaterialButton with icon（可选）
- 卡片圆角和阴影效果

## 3. 技术方案
- 依赖：com.google.android.material:material:1.11.0
- Theme.Material3.DayNight.NoActionBar
- View Binding 保持不变

## 4. 验收标准
- [ ] 界面使用 Material Design 3 组件
- [ ] 支持深色模式
- [ ] UI 层次感和交互反馈提升
- [ ] APK 成功构建

## 5. 依赖
- com.google.android.material:material:1.11.0
