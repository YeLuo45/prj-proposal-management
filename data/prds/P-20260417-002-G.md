# PRD: android-hello 日志框架 Timber

## 1. 概述
为 android-hello 集成 Timber 日志框架，提升调试能力。

## 2. 功能需求

### 2.1 Timber 集成
- 添加 Timber 依赖
- 在 Application 类初始化 Timber
- 替换所有 android.util.Log 调用

### 2.2 日志分级
- DEBUG: 调试信息（用户输入、按钮点击）
- INFO: 应用生命周期事件（Activity 创建/销毁）
- ERROR: 异常信息

### 2.3 Release 构建
- Release 版本自动关闭日志（Timber 自动处理）
- Debug 版本显示日志树

## 3. 技术方案
- 依赖：com.jakewharton.timber:timber:5.0.1
- 自定义 Application 类初始化 Timber
- BuildConfig.DEBUG 判断环境

## 4. 验收标准
- [ ] Timber 正确初始化
- [ ] 关键操作有日志输出
- [ ] Release 构建日志关闭
- [ ] APK 成功构建

## 5. 依赖
- com.jakewharton.timber:timber:5.0.1
