# PRD: android-hello 应用图标 + 闪屏页

## 1. 概述
为 android-hello 添加自定义 App Icon 和 Splash Screen（闪屏页），提升应用专业度。

## 2. 功能需求

### 2.1 自定义 App Icon
- 创建应用图标资源（mipmap）
- 使用 XML 向量图（adaptive icon）支持不同分辨率
- 图标设计：简洁的 Android 机器人 + "HA" 文字

### 2.2 闪屏页（Splash Screen）
- 应用启动时显示 2 秒闪屏
- 中心显示 App Logo 或图标
- 闪屏结束后自动跳转到 MainActivity
- 使用 Kotlin 协程实现延迟跳转

## 3. 技术方案
- Android 12+ SplashScreen API（兼容旧版本用 SplashScreenCompat）
- 图标资源放在 `app/src/main/res/mipmap-*` 目录
- 闪屏页使用专属 Activity（`SplashActivity`）作为启动入口
- `SplashActivity` 延迟 2 秒后跳转到 `MainActivity`，并结束自身

## 4. 验收标准
- [ ] 应用图标在桌面显示正确
- [ ] 启动时显示闪屏页，2 秒后进入主界面
- [ ] `MainActivity` 保持原有功能不变
- [ ] APK 成功构建

## 5. 依赖
- `androidx.core:core-splashscreen:1.0.0`
