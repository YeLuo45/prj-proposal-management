# PRD: android-hello 数据持久化 — SharedPreferences

## 1. 概述
为 android-hello 添加 SharedPreferences 数据持久化功能，保存用户偏好设置（用户名、主题选择、开关设置），并在应用启动时恢复。

## 2. 功能需求

### 2.1 用户偏好存储
- 用户名：首次输入后自动保存，下次启动自动填充
- 主题选择：Light/Dark/System 三选一，实时切换并持久化
- 通知开关：布尔值，记忆用户选择

### 2.2 设置页面
- 独立 SettingsFragment 展示所有偏好项
- 使用 PreferenceFragmentCompat + preference_xml
- 实时预览：修改主题立即生效

### 2.3 数据恢复
- Application.onCreate() 中预加载 SharedPreferences
- MainActivity 启动时读取并恢复上次的用户名和主题

## 3. 技术方案
- SharedPreferences + apply() 异步写入
- PreferenceFragmentCompat 管理偏好 UI
- DataStore 备选（若 SharedPreferences 复杂度上升）
- ViewModel + LiveData 观察偏好变化

## 4. 验收标准
- [ ] 用户名输入后关闭应用再打开，自动填充上次值
- [ ] 主题切换立即生效，重启后保持选择
- [ ] 通知开关状态持久化
- [ ] APK 成功构建

## 5. 依赖
- androidx.preference:preference-ktx:1.2.1
